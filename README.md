## Requirements

- [Scoop](https://github.com/ScoopInstaller/Scoop)

## Usage

```pwsh
## add bucket
$ scoop bucket add <bucket> https://github.com/tbnguyen1407/scoop-bucket

## install manifest
$ scoop install <bucket>/<manifest>
```
